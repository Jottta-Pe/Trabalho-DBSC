🏎️ Checkpoint 1 – EDA Fórmula 1 🏎️

Descrição

Este projeto realiza uma Análise Exploratória de Dados (EDA) com dados de corridas de Fórmula 1.
O objetivo é entender a distribuição dos tempos de corrida e número de voltas, identificar outliers e investigar relações entre variáveis como time, laps e year.
